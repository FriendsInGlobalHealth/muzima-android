Filling Forms For A Patient

Make sure you have downloaded the appropriate form from the server before filling out the form for the patient. If not, go to the forms page. Navigate to the ONLINE tab which will list all the forms available on the server. Select the form you want and then download it.

Once you successfully download the form, return to the home page. Then follow the following steps:

  1. Click the name of the patient you want in the patient list.
  2. Select the appropriate form. That is, the one you downloaded in the FORM FILLING tab within the Data Collection menu on the patient summary page.
  3. This form will pre-populate the patient profile data.
  4. Fill the form for the patient.
  5. Save and sync the form to the server.
Note: Fields in forms marked with an asterisk must be completed in order for the form to be saved as draft or complete.
