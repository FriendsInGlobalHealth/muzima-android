mUzima

mUzima is an android application that renders html5 forms with built in logic and constraints. The application has various prompts including; text, number, location and multimedia which can run in both online and offline situations.

Installation

You will need an Android device to install mUzima. Or install an emulator if you do not have one. mUzima runs on Android version 7.0 and above.

Using the Application

mUzima application will appear in your applications drawer. Select it to launch the application.

Step 1

  • Click NEXT to view the slideshow or SKIP to proceed with the setup.
  • Click the NEXT button for a slideshow of mUzima app features. or, click the SKIP button to continue with mUzima setup

Step 2

  • Read and agree to the mUzima software usage policies by using the checkboxes.
  • Click the ACCEPT ALL button. Then click the NEXT button to continue with the mUzima setup.

Step 3

The first time you log in, you will have three text input prompts.
  • URL field: this is the prompt for the url to the server that the application will be communicating to. e.g "https://demo2.muzima.org".
  • Username: enter the valid username for the user to use the application. This user must be a valid provider on the server.
  • Password: enter the password for the username given in the username prompt.
  • Click LOGIN to authenticate.
After successful authentication you will be taken through a wizard to set up mUzima app.

Step 4

  • mUzima App version 3.x uses guided configuration.
  • Select the program you want to use in the list of programs.
  • Click the NEXT button to continue setting up mUzima.

Step 5

  • mUzima App will be set up.
  • The installation time depends on the amount of data being downloaded from the server.

Step 6

  • The mUzima app will be set up successfully
  • Click the FINISH button to finish the configuration steps.
  • After completing the steps to set up the mUzima app, the mUzima app will display the home page. mUzima software is ready for use.
