mUzima

mUzima est une application Android qui affiche des formulaires html5 avec une logique et des contraintes intégrées. L'application comporte diverses invites, notamment ; texte, numéro, emplacement et multimédia qui peuvent fonctionner dans des situations en ligne et hors ligne.

Installation

Vous aurez besoin d'un appareil Android pour installer mUzima. Ou installez un émulateur si vous n'en avez pas. mUzima fonctionne sur Android version 7.0 et supérieure.

Utilisation de l'application

L'application mUzima apparaîtra dans votre tiroir d'applications. Sélectionnez-le pour lancer l'application.

Étape 1

  • Cliquez sur SUIVANT pour afficher le diaporama ou sur PASSER pour poursuivre la configuration.
  • Cliquez sur le bouton SUIVANT pour un diaporama des fonctionnalités de l'application mUzima. ou, cliquez sur le bouton SKIP pour continuer avec la configuration de mUzima

Étape 2

  • Lisez et acceptez les politiques d'utilisation du logiciel mUzima en cochant les cases.
  • Cliquez sur le bouton ACCEPTER TOUT. Cliquez ensuite sur le bouton SUIVANT pour poursuivre la configuration de mUzima.

Étape 3

Lors de votre première connexion, vous aurez trois invites de saisie de texte.
  • Champ URL : il s'agit de l'invite pour l'URL du serveur avec lequel l'application communiquera. par exemple "https://demo2.muzima.org".
  • Nom d'utilisateur : saisissez le nom d'utilisateur valide permettant à l'utilisateur d'utiliser l'application. Cet utilisateur doit être un utilisateur valide fournisseur sur le serveur.
  • Mot de passe : saisissez le mot de passe du nom d'utilisateur indiqué dans l'invite de nom d'utilisateur.
  • Cliquez sur CONNEXION pour vous authentifier.
Après une authentification réussie, vous serez guidé par un assistant pour configurer l'application mUzima.

Étape 4

  • L'application mUzima version 3.x utilise une configuration guidée.
  • Sélectionnez le programme que vous souhaitez utiliser dans la liste des programmes.
  • Cliquez sur le bouton SUIVANT pour poursuivre la configuration de mUzima.

Étape 5

  • L'application mUzima sera configuré.
  • Le temps d'installation dépend de la quantité de données téléchargées depuis le serveur.

Étape 6

  • L'application mUzima sera configurée avec succès
  • Cliquez sur le bouton TERMINER pour terminer les étapes de configuration.
  • Après avoir terminé les étapes de configuration de l'application mUzima, l'application mUzima affichera la page d'accueil. Le logiciel mUzima est prêt à l'emploi.
