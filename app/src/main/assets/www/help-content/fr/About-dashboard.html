mUzima

Le système de menu inférieur a été utilisé par opposition au tableau de bord dans les versions précédentes du logiciel mUzima. Ce menu est divisé en pages de :

  1. Accueil
  2. Cohortes
  3. Formulaires
  4. Rapports
  1. Accueil
    • Il s'agit de la page qui s'affiche lorsqu'un fournisseur de soins de santé se connecte au programme mUzima.
    • Cette page a un paramètre pour afficher la liste des clients en fonction de la ou des cohortes téléchargées depuis le serveur.
    • Le prestataire de soins de santé peut filtrer cette liste en fonction de la ou des cohortes de patients qu'il souhaite.
    • De même, un prestataire de soins de santé peut rechercher un patient dans la liste des patients en utilisant son nom dans le champ Rechercher un patient.
    • En outre, cette page comporte un bouton d'inscription du patient au cas où le prestataire de soins de santé aurait besoin d'inscrire le patient.
    • Des formulaires incomplets sont fournis dans une zone de cette page pour informer le fournisseur de soins de santé de la présence de ces formulaires.
    • De même, les formulaires non synchronisés se voient attribuer une section sur cette page pour informer le prestataire de soins de santé de l'existence de ces formulaires.
  2. Cohortes
    3. Cette page est divisée en trois onglets.
    1. Onglet TOUT AFFICHER
      2. Cet onglet répertorie tous les groupes de patients dans les onglets TÉLÉCHARGÉS et DISPONIBLES EN LIGNE.
    2. Onglet TÉLÉCHARGÉ
      3. Cet onglet répertorie les groupes de patients téléchargés dans l'application mUzima.
    3. Onglet DISPONIBLE EN LIGNE

      4. Le prestataire de soins de santé peut télécharger la ou les cohortes souhaitées à partir de cet onglet en sélectionnant le cohorte(s) le clic sur l'icône de téléchargement qui s'affiche en haut à droite de l'écran

  3. Formulaires
    4. Cette page est divisée en trois onglets.
    1. Onglet TOUT AFFICHER
      2. Cet onglet répertorie les formulaires dans les onglets TÉLÉCHARGÉS et DISPONIBLES EN LIGNE.
    2. Onglet TÉLÉCHARGÉ
      3. Cet onglet liste les formulaires téléchargés dans le programme mUzima
    3. Onglet DISPONIBLE EN LIGNE

      4. Le prestataire de soins de santé peut télécharger le(s) formulaire(s) requis via cet onglet en sélectionnant le(s) formulaire(s) le clic sur l'icône de téléchargement qui s'affiche en haut à droite de l'écran

  4. Rapports

    5. Cette page répertorie les rapports de performance des agents de santé. Ces rapports peuvent être téléchargés côté serveur.
