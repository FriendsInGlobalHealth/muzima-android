Paramètres dans mUzima

Les paramètres du programme mUzima sont accessibles via la page d'accueil. Cliquez sur l'icône hamburger dans le coin supérieur droit.

Paramètres OpenMRS

Ce paramètre spécifie l'URL du serveur utilisé par mUzima pour accéder aux données. Un utilisateur de mUzima peut modifier cette adresse en cliquant sur le champ d'adresse. Le nom d'utilisateur et le mot de passe ne sont modifiés que sur le serveur.

Paramètres mUzima

Ce sont des paramètres au niveau de l'application.

  • Expiration de la session si le programme n'est pas utilisé. Il s'agit d'un aspect de sécurité de mUzima conçu pour protéger les informations des patients.
  • L'intervalle d'enregistrement automatique du formulaire (minutes) est utilisé pour enregistrer le formulaire sans intervention du professionnel de la santé.
  • Une synchronisation en temps réel qui vous permet de télécharger automatiquement les formulaires remplis sur le serveur.
  • Exiger le numéro de dossier médical rendra le numéro de dossier médical obligatoire s'il est activé.
  • Avertissement des données de formulaire en double si le formulaire a été rempli plus d'une fois ; alerte l'utilisateur lors de l'enregistrement des données.
  • Taille de la police ; l'utilisateur peut choisir la taille de la police entre petite, moyenne et grande.
  • Activer le mode Jour est utilisé pour basculer entre le mode sombre (par défaut) et le mode clair
  • Activer SHR activera/désactivera les dossiers de santé partagés (fonctionnalité de carte à puce) sur l'application mobile
  • Activer le GPS activera/désactivera la collecte d'informations GPS ainsi que la fonction de géocartographie
Paramètres du fournisseur/de l'emplacement

Il s'agit de paramètres au niveau du fournisseur/de l'emplacement

  • Prestataire de rendez-vous par défaut : l'utilisateur connecté sera automatiquement pré-rempli en tant que prestataire de services du patient lors du remplissage d'un formulaire.
  • Gérer les fournisseurs : utilisé pour obtenir d'autres fournisseurs de services du serveur ou les supprimer de l'application mUzima.
  • Lieu de rencontre par défaut : le lieu de rencontre sera utilisé comme lieu de rencontre par défaut lors d'une rencontre et pré-rempli dans un formulaire.
  • Gérer les emplacements : l'utilisateur peut obtenir des emplacements de rencontre à partir du serveur ou supprimer un emplacement de rencontre dans mUzima.
  • Gérer le concept : ce paramètre est utilisé pour obtenir plus de concepts du serveur ou supprimer les concepts de l'appareil.
